# Errors

The StrongBox API uses the following error codes:

`ERR:INVALID_CREDENTIALS`

`ERR:INVALID_PARAMS`

`ERR:NO_EMAIL`

`ERR:PERMISSION_DENIED`

`ERR:NOT_FOUND`

`ERR:USER_CRYPTO_PRIMARY_PUB_KEYS:NOT_FOUND`
